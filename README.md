# Simple Search List
Searching items from API

## For developers
> To clone locally
```
$ git clone git@github.com:luciano01/simple_list_search.git
$ cd simple_list_search
```
> Open the project in an editor, if you use VS Code, follow this
```
$ cd simple_list_search
$ code .
```

> To run, follow this on terminal
```
$ flutter clean
$ flutter pub get
$ flutter run --release
```
